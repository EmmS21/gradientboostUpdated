Monty Hall Simulation¶

Have you heard of the Monty Hall problem? The scenario sounds simple but the right answer seems a bit counter-intuitive to a lot of people. The background behind this is that Monty Hall was the original host of the game show Lets Make a Deal. The Monty Hall problem is based on one of the regular games on the show.

Here is how it goes with our own modification;

Monty starts off by showing you three closed doors, he tells you that there is prize behind each door. One is a car, the other have two old 100 trillion dollar Zimbabwean notes.

Unfortunately these notes will not make you rich, in the slightest. So you would probably want to pick the door with the car. You get to keep whatever is behind the door you pick.

Let's assume you pick Door B, leaving our Door A and C. However, before you open the door the host opens Door A. This door has a 100 trillion Zimbabwean dollar note. Now the host asks you if you want to stick to your original choice, Door B, or switch to the other door that remains closed, Door C.

What would you do?

A lot of people would assume that changing your option would not change your probability of winning. Is this true?

Well it is not.If you switch to Choice C your chances of winning increase to 2/3.

Let's create a Monty Hall simulation to test this.

from random import choice
def single_trial(switch):
    #number of doors
    numDoors = 3
    #choose a random number between 1 and 3
    chosen= random.randint(1,numDoors)
    #assume door 3 was winning choice
    if switch:
        #we reveal door 2 if the player selects door 1 or else we reveal door 1
        revealed = 2 if chosen ==1 else 1
        #these are the doors available
        available = [i for i in range(1,numDoors+1) if i not in (chosen,revealed)]
        chosen = random.choice(available)
    return chosen == 3
#now we simulate multiple attempts 
def multiple_trials(trials,switch):
    numDoors = 3
    numWins = 0
    for i in range(trials):
        if single_trial(switch):
            numWins +=1
    return numWins
def print_string(trials,switch):
    numDoors = 3
    if switch == False:
        return 'Probability of wins without switch out of {} attempts is {:.4f}'.format(trials,multiple_trials(trials,switch)/trials)
    else:
        return 'Probability of wins with switch out of {} attempts is {:.4f}'.format(trials,multiple_trials(trials,switch)/trials)
#1000 attempts without switching
print_string(100000,False)
